# Travel Insights Dashboard

A full-stack analytics dashboard built with **Flask (Python)** and **React (Vite)** that analyzes airline flight data and visualizes revenue insights.

This project demonstrates end-to-end system design including:
- REST API development
- Database modeling
- Analytics computation
- Frontend data visualization
- Search, sorting & filtering
- Data-driven dashboard UI

---

## Features

### Backend (Flask API)
- Retrieve all flights
- Filter flights by airline
- Calculate total revenue
- Compute average ticket price
- Revenue breakdown by airline
- Analytics endpoint for dashboard metrics

### Frontend (React)
- Full-screen responsive dashboard
- KPI cards (Flights, Revenue, Avg Price)
- Flight search by airline
- Price sorting (Ascending / Descending)
- Scrollable flights table
- Top airline revenue panel
- Modern gradient UI
